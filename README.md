# GROCIFY
` An ecommerce platform for a grocery company `

## Description
An E-commerce platform to buy groceries with a paymetn gateway

## Features

* ->  Add items to the cart
 * -> Remove items from the cart
 * -> Place Order
 * ->  Browse Products
* -> Animations made using Framer-Motion
* -> Payment Gateway
* -> Toast Icon
* -> automated email system using email Js
## Tech Stack 
* -> React Js
* -> Express Js
* -> crypto
* -> Razorpay for Payment gateway
* -> Tailwindcss
* -> FontAwesome
* -> Framer Motion
* -> Razorpay auth
* -> EmailJS
